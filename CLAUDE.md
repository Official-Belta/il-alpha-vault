# IL Alpha Vault

## Project
Uniswap V4 hook-based automated LP vault.
Compare fee yield vs IL cost using options pricing in real-time.
Only maintain LP in positive expected value ranges.

## Tech Stack
- Solidity (Foundry)
- Uniswap V4 hooks
- Python (backtesting)

## Key Functions (from BELTA)
- _calculateIL
- _positionValueUSDC
- _feesToUSDC
