1단계: "깨진 것을 고쳐라" (2021.06)

Uniswap V3가 출시되면서 LP 포지션이 NFT가 됨 → 모든 프로토콜의 유동성 마이닝이 깨짐.

문제: Uniswap V2: LP 토큰 = ERC-20 → 스테이킹 가능 ✅ Uniswap V3: LP 포지션 = NFT → 스테이킹 불가 ❌ → 모든 프로토콜이 "V3에서 LP 마이닝 어떻게 하지?" 고민 G-UNI 해결: V3 NFT → ERC-20 래핑 → 스테이킹 가능 ✅ + 자동 리밸런싱 + 수수료 자동 복리 → "토큰 런칭하면 G-UNI 쓰세요"가 업계 표준이 됨

런칭 Day 1에 Instadapp과 동시 출시. INST/ETH 풀에 APY 2,200%를 제공하며 즉각적인 주목을 받음.

→ 교훈: 첫 고객을 런칭 전에 확보하라. 빈 프로토콜은 아무도 안 쓴다.

3단계: 구조적 수요 확보 (MakerDAO, 2021.09)

MakerDAO가 G-UNI DAI/USDC 토큰을 담보로 승인. 당시 MakerDAO 담보의 54%가 USDC 단일 자산이었고, G-UNI는 이를 생산적 자산으로 전환하는 방법을 제공.

구조적 수요 루프: 사용자가 DAI 필요 → G-UNI DAI/USDC를 MakerDAO에 담보로 예치 → DAI 발행 → G-UNI 수요 증가 → TVL 증가 → 더 깊은 유동성 → 더 많은 사용자 → 반복

→ 교훈: "담보로 승인받기" = 마케팅보다 강력한 성장 엔진.

V4 Dynamic Fee Hook — 기술 분석

Arrakis 방식

beforeSwap에서 수수료 조정

1. CEX 가격과 풀 가격 비교

2. 차이 크면 (차익거래) → 수수료 ↑↑

3. 재고 불균형 시 → 밸런싱 방향 수수료 ↓

장점: LP 유지 → 수수료 계속 수취

단점: IL 노출 유지 (감소만, 제거 아님)

의존: 오프체인 CEX 가격 피드

VS
IL Alpha 방식

afterSwap에서 vol 업데이트

1. EWMA vol oracle 업데이트

2. feeYield vs ilCost 비교

3. ilCost > feeYield → LP 완전 제거

장점: IL 노출 = 0 (완전 제거)

단점: LP OFF 시 수수료 수취 0

의존: 온체인 EWMA만 (탈중앙)

Arrakis V4 Hook 작동 흐름: 스왑 요청 → beforeSwap hook 실행 │ ├─ CEX 가격 조회 (오프체인) ├─ 풀 가격과 비교 ├─ 차이 = 차익거래 가능성 │ ├─ 차이 크면 → 수수료 50bps → 200bps+ ├─ 차이 작으면 → 수수료 5bps (정상) └─ 재고 불균형 → 밸런싱 방향 할인 결과: 독성 플로우(차익거래) 억제 일반 거래는 계속 수수료 수취 LP는 항상 풀에 남아있음 IL Alpha Hook 작동 흐름: 스왑 완료 → afterSwap hook 실행 │ ├─ EWMA vol oracle 업데이트 (온체인) ├─ volume EWMA 업데이트 ├─ feeYield vs ilCost 계산 │ ├─ feeYield > ilCost → LP ON (유지) └─ feeYield < ilCost → LP OFF (제거) 결과: 위험 시 LP 완전 철수 IL 노출 = 0 수수료 수취도 0 (LP OFF 동안)

1. 타이밍 = 전부

2. Day 1 고객

❌ 안 따라할 것

1. 18개월 무매출

$SPICE 3년 미출시 → 신뢰 파괴. 안 할 거면 언급 자체 금지.

3. 인센티브 의존 TVL

4. TVL = 허영 지표

6. moat 없는 선점

위협 평가: Arrakis가 우리를 카피하면?

시나리오가능성영향이유
Arrakis가 "LP 제거 모듈" 추가 낮음 중간 B2B 고객이 유동성 보장을 원함. LP 끄는 건 B2B DNA에 반함.
Arrakis가 B2C 패시브 상품 런칭 중간 높음 가능하지만 현재 B2B에 집중. 리테일 UX 빌드 필요.
Arrakis가 우리를 인수 제안 매우 낮음 긍정적 $4M 펀딩 규모에서 인수는 비현실적. 하지만 파트너십은 가능.
핵심 결론: Arrakis는 B2B, 우리는 B2C. 같은 시장이 아닙니다. Arrakis의 초창기 플레이북 (타이밍 선점 + 첫 고객 확보 + 구조적 수요 창출)은 배울 가치가 있지만, 그들의 실수 (18개월 무매출, 토큰 미출시, 인센티브 의존)는 피해야 합니다.
